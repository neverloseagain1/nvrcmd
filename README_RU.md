# Справочник NVRCMD 🚀

> [!NOTE]
> 🌐 **Language / Язык:**
> * 🇺🇸 [English](README.md)
> * 🇷🇺 **Русский** (Текущий)

Интерактивная мультиязычная терминальная утилита, созданная для новичков, чтобы быстро находить и запоминать базовые и продвинутые команды Linux. Она автоматически определяет ваш дистрибутив и имеет удобный интерфейс с прокруткой.

## ✨ Особенности

* 📦 **Автоопределение ОС:** Автоматически читает `/etc/os-release` и подтягивает команды именно под вашу систему (Arch Linux, Ubuntu, Debian, Fedora, NixOS, Gentoo).
* 🔄 **Локализация на лету:** Нажмите кнопку `l` для мгновенного переключения интерфейса и описаний между русским и английским языками.
* 💾 **Сохранение настроек:** Программа запоминает ваш выбор языка, создавая скрытый конфиг `~/.config/nvrcmd.json`.
* 🛡️ **Чистый терминал:** Открывается в отдельном буфере экрана (как `htop` или `vim`) и полностью стирает себя из консоли после выхода, не забивая историю.
* 💡 **Полезные примеры:** Каждая команда снабжена реальным и безопасным примером использования справа от описания.

## 📦 Поддерживаемые дистрибутивы

NVRCMD автоматически распознает и предоставляет специализированные команды для следующих семейств Linux:
* 🔵 **Arch Linux** (включая совместимость с Void Linux)
* 🟠 **Ubuntu / Debian / Linux Mint**
* 🟢 **Fedora Linux**
* ⚪ **NixOS** (с декларативным синтаксисом и гидами по `nix-shell`)
* 🟣 **Gentoo Linux** (с компиляцией через `emerge` и настройкой USE-флагов)

## 📥 Установка

Вы можете легко скомпилировать и установить утилиту в систему с помощью готового интерактивного скрипта.

```bash
# Перейдите в папку с проектом
cd nvrcmd

# Запустите интерактивный инсталлятор
./install.sh
```

## ⌨️ Управление

* `↑ / ↓` или `j / k` — Перемещение по списку команд вверх и вниз
* `l` — Переключение языка интерфейса (RU / EN)
* `q` или `Ctrl + C` — Безопасный выход из справочника
